# タスク完了時のチェックリスト

## 必須実行コマンド（appディレクトリで実行）

1. **コードフォーマット**

   ```bash
   pnpm format
   ```

2. **リントチェック**

   ```bash
   pnpm lint
   ```

3. **ビルド確認**

   ```bash
   pnpm build
   ```

4. **テスト実行**（テストが存在する場合）
   ```bash
   pnpm test
   ```

## 確認事項

- [ ] すべてのTypeScriptエラーが解決されている
- [ ] Biomeのリント・フォーマットエラーがない
- [ ] ビルドが成功する
- [ ] 既存のテストが通る（存在する場合）
- [ ] 新機能にテストを追加した（必要に応じて）

## コミット前の注意

- 機能するコードを破壊していないか確認
- 不要なconsole.logやデバッグコードを削除
- 環境変数やシークレットをコミットしていないか確認
