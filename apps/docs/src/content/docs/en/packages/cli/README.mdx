# @himorishige/hatago-cli

Command-line utilities for developing Hatago MCP Hub.

> Note: End users should use `@himorishige/hatago-mcp-hub` instead.

## Install

```bash
npm install -g @himorishige/hatago-cli
# or
npx @himorishige/hatago-cli --help
```

## Commands (examples)

- `hatago serve` — start a local hub for testing
- `hatago mcp list|add|remove` — manage child MCP servers
- `hatago config show|get|set|reset` — inspect config during development

## Config files

Stored under `~/.hatago/` for dev workflows.
