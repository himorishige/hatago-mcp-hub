## Config

```json title="hatago-config.json"
{
  "$schema": "https://raw.githubusercontent.com/himorishige/hatago-hub/main/schemas/config.schema.json",
  "version": 1,
  "logLevel": "debug",
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "."],
      "tags": ["essential", "dev"]
    },
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": { "GITHUB_TOKEN": "${GITHUB_TOKEN}" },
      "tags": ["vcs", "dev"]
    },
    "database": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-postgres"],
      "env": { "DATABASE_URL": "${DATABASE_URL:-postgresql://localhost/dev}" },
      "tags": ["database", "dev"]
    }
  }
}
```

## Start

```bash
# Auto-reload with nodemon
npx nodemon --exec "hatago serve --tags dev" --watch hatago.config.json

# Or run directly
hatago serve --tags dev
```

## Tip

Use tag sets (`dev`, `staging`, `production`) and `--tags` to switch environments.
