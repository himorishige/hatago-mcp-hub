## Prerequisites

- Node.js 20+

## 1. Initialize

```bash
npx @himorishige/hatago-mcp-hub init
```

## 2. Configure Client

Claude Code `.mcp.json`:

```json
{
  "mcpServers": {
    "hatago": {
      "command": "npx",
      "args": [
        "@himorishige/hatago-mcp-hub",
        "serve",
        "--stdio",
        "--config",
        "./hatago.config.json"
      ]
    }
  }
}
```

## 3. Run

```bash
npx @himorishige/hatago-mcp-hub serve --stdio --config ./hatago.config.json
```
