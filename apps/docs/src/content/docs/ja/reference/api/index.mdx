# APIリファレンス

Hatago MCP HubのAPIリファレンスです。

## Hub API

### createHub

```typescript
function createHub(config: HubConfig): Hub;
```

Hubインスタンスを作成します。

### Hub.start()

```typescript
async start(): Promise<void>
```

Hubを起動します。

### Hub.listTools()

```typescript
async listTools(): Promise<Tool[]>
```

利用可能なツール一覧を取得します。

## Server API

### startServer

```typescript
async function startServer(options: ServerOptions): Promise<void>;
```

サーバーを起動します。

## 内部リソース

### `hatago://servers`

現在接続しているサーバー一覧を JSON で返します。`id` / `status` / `type` に加え、`tools` / `resources` / `prompts` の配列を含みます。
