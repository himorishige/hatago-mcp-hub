## インストール方法

### すべてのパッケージを使用する場合

```bash
npm install @himorishige/hatago-mcp-hub
```

### 特定のパッケージのみ使用する場合

```bash
# Hubコアのみ
npm install @himorishige/hatago-hub

# サーバー実装のみ
npm install @himorishige/hatago-server

# 型定義のみ
npm install @himorishige/hatago-core
```
