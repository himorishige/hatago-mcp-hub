# 基本概念

Hatago MCP Hubの基本的な概念と仕組みについて説明します。

## Hub

Hubは複数のMCPサーバーを統合管理する中核コンポーネントです。

## Server

個々のMCPサーバー実装です。ローカル、NPX、リモートサーバーをサポートします。

## Transport

通信層の実装。STDIO、HTTP、SSE、WebSocketに対応します。

## Core

共通の型定義とインターフェースを提供します。
